# 파이썬 멀티프로세싱 큐(Queue) 실습

이번 과제는 파이썬 `multiprocessing` 모듈의 `Queue` 클래스를 사용하여 데이터를 큐에 넣고 빼면서 큐의 기본 동작 원리를 이해하는 실습 과제이다.

## 이번 프로젝트를 하며 배운 점들

**1. `multiprocessing.Queue` 객체 생성 및 활용**
`multiprocessing.Queue()`를 선언하여 데이터를 담을 수 있는 큐 객체를 생성하고 다루는 기초적인 방법을 익혔다.

**2. `.put()` 메서드를 통한 데이터 삽입 (Push/Enqueue)**
리스트에 담긴 여러 색상 데이터('red', 'green', 'blue', 'black')를 반복문을 통해 큐에 순차적으로 집어넣는(`put()`) 동작을 구현했다. 

**3. `.get()` 메서드를 통한 데이터 추출 (Pop/Dequeue)**
큐에 들어있는 데이터를 하나씩 꺼낼 때(`get()`), 처음에 집어넣었던 순서 그대로('red' -> 'green' -> 'blue' -> 'black') 출력되는 것을 확인했다. 이를 통해 가장 먼저 들어간 데이터가 가장 먼저 나오는 큐의 핵심 특성인 **선입선출(FIFO, First-In-First-Out)** 구조를 확실하게 체감했다.
