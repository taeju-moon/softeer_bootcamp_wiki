## mtcars 데이터셋 분석하기

### 1. 기본 함수
- head: 상위 5개 행 출력
- tail: 하위 5개 행 출력
- shape: (행, 열) 반환
- info, dtypes: 각 열의 기본 정보 반환
- describe: 각 열의 기본 통계값들 출력ㄹ (count, mean  등)

### 2. gear transmission 조합
- unique: 각 컬럼의 unique한 종류 수를 확인할 수 있음
- crosstab: pandas 기본 탑재 라이브러리로 컬럼 조합에 따른 테이블 구성
- groupby: 해당 열들로 groupby 해서 size()를 통해 각 groupby 행의 개수를 구한다. (조합 구하기)

### 3. 데이터셋 컬럼들 히스토그램 그리기
- subplots: 플롯 안에 여러 테이블들을 그려 넣을 수 있음
- enumerate로 순회하면서 각 테이블들을 plot
- plot kind='hist'는 plot의 형식이 histogram임을 의미함

### 4. Scatter Plot
- plt.scatter() 함수로 산점도 그릴 수 있음
- plt.grid()를 통해 배경 그리드 작성

### 4. 상관계수 표 출력
- df.corr()를 통해 상관계수 테이블을 만들 수 있음
- x축의 컬럼과 y축의 컬럼이 어떻 상관관계가 있는지 파악 가능
